## 🛠 Công nghệ sử dụng
- **Ngôn ngữ:** Java 25
- **Framework:** Spring Boot 4.0.5, Spring Security, JPA, Thymeleaf
- **CSDL:** MySQL 8.0
- **Giao diện:** Bootstrap 5, FontAwesome, Google Fonts

## ⚙️ Hướng dẫn cài đặt cho thành viên
1. **Clone dự án:** `git clone https://github.com/Okeydokey8525/Java_DungCuHocTap.git`
2. **Cài đặt CSDL:** - Mở MySQL Workbench.
   - Chạy toàn bộ file trong thư mục `database/database_setup.sql`.
3. **Cấu hình mật khẩu:** Mở `src/main/resources/application.properties`, sửa `spring.datasource.password` theo mật khẩu máy bạn.
4. **Chạy ứng dụng:** Chạy file `VanPhongPhamApplication.java` và truy cập `http://localhost:8080`.
